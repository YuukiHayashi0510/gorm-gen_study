## 特徴

- .ent に似ている
- gorm に慣れていると、生成が楽
- 既存の DB に定義がある場合、かなり楽にモデルを生成できる
- アソシエーションは流石に読み取ってくれないので、手動で定義を追加する必要あり
  - 0~作るよりはかなり楽だとは思う

## Gorm Gen の使い道

- 既存の DB に定義が存在する時
- Gorm に慣れている人が多い時
